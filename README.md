# DeathRecall

一个简单的 Minecraft 1.20.1 Forge Mod，允许玩家通过指令回到上一次死亡地点。

## 功能

- **`/back` 指令**：将玩家传送到最后一次死亡的位置。
- **跨维度支持**：支持在主世界、下界、末地等不同维度之间进行传送。
- **无需权限**：普通玩家即可使用，无需 OP 权限。
- **持久化记录**：使用原生的死亡位置记录，即使重新进入游戏也能正确回到位置。

## 安装要求

- **Minecraft 版本**: 1.20.1
- **Forge 版本**: 推荐 47.4.16 或更高版本

## 使用说明

在游戏中输入以下指令即可：
```text
/back
```
